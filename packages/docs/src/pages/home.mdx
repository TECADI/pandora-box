## Como usar

<Divider space={'8px 0'}/>

### Pandora.box - Design System


Para utilizar os componentes do pandora.box, basta instalar a biblioteca em seu projeto e importar os componentes que deseja utilizar. Veja abaixo como instalar a biblioteca:


```bash

npm install @pandora-box-tecadi/design-system
```


```bash

yarn add @pandora-box-tecadi/design-system
```
